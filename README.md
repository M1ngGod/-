# -
riskbird.com 风鸟备案批量查询包括下属单位，自定义控股

默认不加-o参数的话会自动保存到当前目录result下面
加-q的话就默认也收集下属单位了，如果不加-q参数默认都是搜索指定单位
其他操作-h即可 【有帮助的话请给个小星星】

自己创建一个config.txt，用来放cookie。
